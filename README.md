# web-attack-log-analyzer

웹 공격 시나리오를 재현하고 서버 로그를 분석하여 침해 흔적과 대응 방안을 정리하는 프로젝트입니다.

## 프로젝트 목적

본 프로젝트는 실습용 웹 서버를 직접 구축하고, 로그인 Brute Force, SQL Injection, XSS 등 대표적인 웹 공격 시나리오를 재현한 뒤, 웹 서버 로그와 애플리케이션 로그를 분석하여 공격자의 의도와 침해 흔적을 파악하는 것을 목표로 합니다.

## 주요 기능

- 실습용 웹 서버 구축
- 정상 요청 로그 수집
- 웹 공격 시나리오 재현
- 웹 서버 및 애플리케이션 로그 분석
- 공격 패턴 및 의심 요청 탐지
- 침해 분석 리포트 작성
